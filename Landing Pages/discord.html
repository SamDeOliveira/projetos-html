Temas pra Discord 

Os temas são feitos no Photoshop, upados na nuvem, e importados no código em css. 
 Pra usar o tema é só colar o meu código no progama.

Tutorial 
Passo 1: instalar o Better Discord 
 Um 'mod' que permite instalação de plugins e temas para o discord.


 https://betterdiscord.app/
 


Passo final: Usar o Css Personalizado 
 Após Instalado, novas configurações aparecerão. 
 Abra as configurações do Discord e desça até
 achar a opção 'Css Personalizado'.  

agora é só colar o meu código css e pronto ;)
